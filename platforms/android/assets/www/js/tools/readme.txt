开发过程中可能涉及的插件列表:
序号      名称                      路径                              功能

1     auto-focus.js             当前目录                     提供输入框的自动聚焦，弹出手机上的键盘

2     imageCacheFactory         当前目录                     用于图片的缓存，主要用于list+点击进行子页面的形式，
                                                            子页面会用到list主页缓存的图片，避免同一图片的多次下载，
                                                            http://mcgivery.com/preload-images-ionic/

3     ionic-image-lazy-load.js  lib/ion-image-lazy-load/    对于还未显示到的图片不进行加载，推迟到翻阅到的时候进行加载
                                                            用于大量图片加载导致速度过慢的情况
                                                            https://github.com/paveisistemas/ionic-image-lazy-load
